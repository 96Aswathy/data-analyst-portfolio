Sales Dashboard - Power BI

Description

This project is a Sales Analytics Dashboard created using Power BI. It provides insights into sales performance by analyzing revenue trends, product categories, regions, and customer segments.

Tools Used

- Power BI
- Excel/CSV Dataset
- DAX

Dashboard Features

- Total Sales KPI
- Total Profit Analysis
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Segment-wise Sales Analysis
- Top Performing Products

Objective

To analyze sales data and create an interactive dashboard that helps understand business performance and identify key sales trends.
