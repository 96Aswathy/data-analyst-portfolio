# HR Analytics Dashboard

## Project Overview
This project analyzes HR data using SQL, MySQL, and Power BI to understand employee demographics, attrition, salary, job satisfaction, and workforce trends.

## Tools Used
- MySQL
- SQL
- Power BI
- Microsoft Excel

## Dataset
- 1,423 employee records
- HR Analytics dataset

## Dashboard Features
- HR Overview
- Employee Insights
- Attrition Analysis

## Key KPIs
- Total Employees
- Attrition Count
- Attrition Rate
- Average Monthly Income
- Average Age
